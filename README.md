# LinkWake

Phase 0 fun-yield prototype for the browser-native page-as-level game.

The current build has the v1 level-script schema locked, validates three placeholder levels at module load, and renders a PixiJS three-tile preview for the longform, reference, and commerce archetypes. Checkpoint 3 is now wiring the first interactive SURGE runtime; backend, accounts, compiler, extension, audio, HUD polish, and sharing remain intentionally out of scope for Phase 0.
